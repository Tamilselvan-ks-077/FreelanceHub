# Django Project

This is a clean, default Django project template.

## 🚀 How to Run the Project

### 1. Activate the virtual environment
```bash
source venv/bin/activate
```

### 2. Start the development server
```bash
python manage.py runserver
```

### 3. Visit the site
Open [http://127.0.0.1:8000/](http://127.0.0.1:8000/) in your browser to see Django's default welcome page.
